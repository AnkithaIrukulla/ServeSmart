# 🚀 ServeSmart

ServeSmart is a web-based service management system designed to streamline operations and improve workflow efficiency.

## ✨ Features
- User Authentication (Login/Register)
- Dashboard Management
- Profile Management
- File Upload System

## 🛠️ Tech Stack
- PHP
- MySQL
- JavaScript
- CSS
- XAMPP

## ⚡ How to Run
1. Clone the repository
2. Move project to `htdocs`
3. Start Apache & MySQL in XAMPP
4. Open browser:
   http://localhost/ServeSmart
